Capacity Formula Breakdown

# OFDM Capacity per channel:
C = N_sc × log₂(M) × R_c × f_s

Where:
N_sc = subcarriers per channel
M = QAM order
R_c = coding rate
f_s = subcarrier spacing (= symbol rate per carrier)

Total (OFDM channels):
C_total = C × N_ofdm_channels
